Documentation

What if we could stop time?

This query is universal to the human experience. For me it's most common when listening to music, where I yearn to freeze a texture and fall into it forever. This is an exploration of that auditory paradise.

Kaleidophone is designed for deep thinking. With binaural beats at 8 Hz and a reference frequency of 432 Hz, it's perfect for any contemplative occasion. Explore its ambient soundscapes by manipulating time. Rewind, scrub, freeze, or adjust the playback speed of unique songs each use.

Please enjoy wherever it takes you.

System Requirements

Kaleidophone is an audio toy best experienced with headphones.

Real-time synthesis is an extremely CPU-intensive process. Kaleidophone will attempt to run on any machine that meets its minimum requirements; however, users may experience stuttering or drop-outs when running below the recommended requirements.

Minimum System Requirements

CPU SSE2 capable (64-bit)
RAM 2 GB

Recommended System Requirements

CPU 3.5 GHz Quad-Core (64-bit)
RAM 4 GB

Browser Support

The HTML5 build of Kaleidophone is intended to be used in a recent version of a Chromium-based browser, including Google Chrome, Microsoft Edge, and Opera. Browser extensions that alter page behavior are not recommended. Other browsers may suffice if they support modern JavaScript features and the current Web Audio API specification.

How to Use

Interact to begin. Move forward and backward to scrub through time. Turn to pan and calibrate its kaleidoscopic visualizer. Play and rewind at adjustable speeds. Randomize at any time to hear something new.

Accessibility

Kaleidophone is developed in accordance with WCAG (Web Content Accessibility Guidelines) 2.1 at Level AA conformance. Its menus are fully operable with a keyboard and screen readers such as NVDA.

Graphical Description

Each tone that the kaleidophone generates corresponds to a colored square drawn on its visualizer. It exhibits a radial symmetry which renders a colorful mandala of patterns that dance and evolve with its music.

Photosensitivity Warning

Kaleidophone is designed to conform to WCAG 2.1 SC 2.3.1 (Three Flashes or Below Threshold). However, beware that its visualizer can produce colorful and contrasting patterns that may be harmful. Users with photosensitivity are encouraged to take precaution before use.
